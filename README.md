# 🎨 Color Blindness Simulator

A web application that allows users to upload an image and preview how it appears under different types of color vision deficiencies.

## Features

- Upload an image
- View original image
- Simulate:
  - Protanopia
  - Deuteranopia
  - Tritanopia
  - Achromatopsia
- Before & After comparison slider
- Image information
  - File name
  - File size
  - Dimensions
- Responsive Glassmorphism UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
- HTML5 Canvas API

## How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Upload an image.
4. Select a color blindness mode.
5. Compare the original and simulated image.
